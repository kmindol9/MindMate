# MindMate

### AI 기반 미래 불안 분석 및 방향성 제시 시스템

---

## Project Overview

학생들은 제한된 경험과 환경 속에서
미래에 대한 막연한 불안과 방향성의 혼란을 겪는다.

**MindMate**는 이러한 문제를 해결하기 위해,
사용자의 감정을 분석하고 문제를 구조화하여
**데이터 기반의 객관적인 피드백을 제공하는 AI 시스템**이다.

---

## Problem Statement

- 청소년 및 대학생의 불안과 우울 증가
- 제한된 경험으로 인한 왜곡된 자기 인식
- 기존 상담 시스템의 접근성 및 객관성 부족

핵심 문제는 **“자신의 상태를 객관적으로 인식하지 못하는 것”**

---

## Solution Approach

본 시스템은 다음과 같은 흐름으로 작동한다:

1. 사용자 고민 입력
2. 감정 분석 수행
3. 문제 유형 분류
4. 문제 구조화
5. 데이터 기반 피드백 생성

단순 상담이 아닌 **분석 + 구조화 + 데이터 기반 의사결정 지원 시스템**

---

## System Architecture

```
User → AI Counseling System → AI Module / Data Source → Result
```

- **User**: 고민 입력
- **System**: 분석 및 구조화
- **AI Module**: 자연어 기반 피드백 생성
- **Data Source**: 통계 및 사례 데이터

---

## Key Features

- 감정 분석 (Emotion Analysis)
- 문제 분류 (Problem Classification)
- 문제 구조화 (Problem Structuring)
- 데이터 기반 피드백 (Data-based Feedback)
